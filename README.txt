CARTE DE VISITE VIRTUELLE PREMIUM — BAILLY IMMOBILIER

MISE EN LIGNE GRATUITE AVEC GITHUB PAGES

1. Va sur github.com et crée un compte.
2. Clique sur “New repository”.
3. Nom du dépôt : carte-bailly-immobilier
4. Choisis “Public”, puis “Create repository”.
5. Clique sur “Add file” > “Upload files”.
6. Dépose les 5 fichiers suivants :
   - index.html
   - styles.css
   - script.js
   - logo-bailly.png
   - thomas-bailly.vcf
7. Clique sur “Commit changes”.
8. Va dans Settings > Pages.
9. Dans Source, choisis “Deploy from a branch”.
10. Branche : main ; dossier : /(root), puis Save.

Après quelques minutes, l’adresse sera du type :
https://TON-IDENTIFIANT.github.io/carte-bailly-immobilier/

IMPORTANT
- Ne renomme pas les fichiers.
- Mets tous les fichiers à la racine du dépôt.
- Quand le lien est créé, teste les boutons appel, e-mail, WhatsApp et ajout aux contacts depuis ton téléphone.
